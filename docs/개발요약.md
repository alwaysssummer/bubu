1. 목적 : 부부간의 월별 지출과 수익을 예상하고, 실제 집행을 1만원단위로 기록하고 공유하는 초간단가계부와 할일 공유
2. 기능 
 2-1. 실행
   -실제 가계부,태그(항목)으로 간단하게 수익/지출만 1만원 단위로 입력하고, 현재 잔고만 보여줌
 2-2. 예상
   -매월 예산 체크 목록 : 매월 반복 또는 예상되는 지출/수익을 기록하고, 집행과정을 체크 표시해서 이번달 예산 집행 과정을 확인하고 예상할수있음. (반복기능추가)
 2-2. 할일목록
   -서로에게 할일을 요청하면, 언제까지 하겠다는 날짜를 입력해서 알려주고, 처리결과를 간단히 메모
3.디자인
  - 모바일 버전으로 최대한 기능중심의 직관적이고 실용적으로 / 불필요한 디자인은 최소화
  - 샤드cn 디자인의 흑/백 노트 느낌의 심플하고 세련된 디자인으로
4. 개발스텍
 - 슈파베이스/ 버셀 배포
 - next.js 최신 버번으로로
